# CHG-020 카테고리 정렬 및 경매 취소요청 코드

## 변경 목적

- 화면 표시 순서를 상품 카테고리 `1~10`, 서비스 카테고리 `11~15`로 통일한다.
- 판매자가 취소 사유를 제출한 뒤 관리자 판단을 기다리는 경매 상태 `AUCC0006(취소요청)`을 추가한다.

## 변경하지 않는 값

- `CATEGORY.CAT_SN`은 상품·서비스·제공자 권한 등에서 참조하는 내부 식별번호이므로 변경하지 않는다.
- 기존 `AUCC0005(취소)`는 관리자 승인 또는 운영 조치로 확정된 최종 취소 상태다.

## 실행 순서

1. 개발 DB에서 [증분 SQL](./20260720_CHG-020_category_sort_and_auction_cancel_request.sql)을 실행한다.
2. SQL 내부 `SELECT` 결과를 확인한다.
3. 결과가 맞으면 `COMMIT`, 이상하면 `ROLLBACK`한다.

## 상태 흐름

`진행/종료 → 취소요청(AUCC0006) → 승인 시 취소(AUCC0005)`

반려 시 이전 경매 상태 복구 기준은 거래·경매 담당 API 구현과 맞춰 결정한다.
